## Getting Started

### What is AI Rental Tools?

AI Rental Tools is a comprehensive property management platform designed for:
- **Individual Landlords** - Manage one or more rental properties
- **Multi-Property Owners** - Handle portfolios of rental units
- **Property Management Companies** - Manage multiple clients and their properties

### System Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge - last 2 versions)
- Internet connection
- Mobile device or desktop computer

### Creating Your Account

1. Visit the AI Rental Tools website
2. Click **Sign Up** or **Get Started**
3. Choose your account type:
   - **Single Landlord** - For managing 1-10 properties
   - **Multi-Property Owner** - For landlords with larger portfolios
   - **Property Manager** - For managing multiple client businesses
4. Enter your information
5. Verify your email address
6. Complete the onboarding wizard

## Registration & Onboarding

### Registration Flow Types

#### Type 1: Single Landlord
**Best for:** Individual landlords with 1-3 properties

**Steps:**
1. Enter your personal information
2. Add your first property details
3. Set up units within your property
4. Optionally add existing tenants
5. Complete setup and access your dashboard

#### Type 2: Multi-Property Owner
**Best for:** Landlords with 4+ properties

**Steps:**
1. Create your account
2. Set up your portfolio
3. Add multiple properties
4. Configure units
5. Import or add tenants
6. Customize settings

#### Type 3: Property Management Company
**Best for:** Professional property managers

**Steps:**
1. Create your management company profile
2. Set up your first client business
3. Add client properties
4. Configure units and tenants
5. Customize billing and reporting
6. Invite team members

### First-Time Setup Wizard

After registration, you'll be guided through a setup wizard:

1. **Welcome Screen** - Overview of features
2. **Property Setup** - Add your first property
3. **Unit Configuration** - Define rental units
4. **Tenant Management** - Add existing tenants (optional)
5. **Payment Setup** - Configure rent collection
6. **Review & Launch** - Confirm settings and start

**Tip:** The wizard saves your progress automatically. You can pause and resume later.

## Managing Properties

### Adding a Property

1. Click **Properties** in the sidebar
2. Click **Add Property** button
3. Fill in property details:
   - Property name
   - Full address
   - Property type (Single-family, Multi-family, Apartment, etc.)
   - Number of units
   - Purchase date (optional)
   - Property value (optional)
4. Click **Save**

### Property Details

Each property has:
- **Overview** - Key information and statistics
- **Units** - List of all rental units
- **Tenants** - Current and past tenants
- **Financials** - Income and expenses
- **Documents** - Leases, contracts, inspections
- **Maintenance** - Work orders and history

### Managing Units

#### Adding Units

1. Open a property
2. Click **Units** tab
3. Click **Add Unit**
4. Enter unit details:
   - Unit number/name
   - Bedrooms and bathrooms
   - Square footage
   - Monthly rent
   - Security deposit
   - Amenities
5. Click **Save**

#### Unit Status

- **Vacant** - Available for rent
- **Occupied** - Has active tenant
- **Under Maintenance** - Being repaired
- **Unlisted** - Not available for rent

### Bulk Import

Import properties from:
- CSV file (standard format)
- Buildium export
- AppFolio export
- Yardi export

**Steps:**
1. Click **Import** button
2. Select data source
3. Download template
4. Fill in your data
5. Upload file
6. Review and confirm

## Managing Tenants

### Adding a Tenant

1. Click **Tenants** in sidebar
2. Click **Add Tenant**
3. Enter tenant information:
   - Full name
   - Email address
   - Phone number
   - Emergency contact
   - Move-in date
4. Assign to unit
5. Set up lease details
6. Click **Save**

### Tenant Portal

Each tenant gets access to a self-service portal where they can:
- View lease details
- Pay rent online
- Submit maintenance requests
- View payment history
- Upload documents
- Message you securely

#### Inviting Tenants

1. Open tenant profile
2. Click **Send Portal Invitation**
3. Tenant receives email with link
4. They create password and access portal

### Lease Management

#### Creating a Lease

1. Open tenant profile
2. Click **Leases** tab
3. Click **New Lease**
4. Enter lease details:
   - Start and end dates
   - Monthly rent amount
   - Security deposit
   - Lease type (Fixed-term, Month-to-month)
   - Special terms
5. Upload lease document (optional)
6. Click **Create Lease**

#### Lease Renewal

30-60 days before expiration, you'll receive renewal alerts:
1. Click alert or open lease
2. Click **Renew Lease**
3. Adjust terms if needed
4. Send renewal offer to tenant
5. Track tenant response
6. Finalize renewal

## Maintenance Requests

### Tenant-Submitted Requests

Tenants can submit requests through their portal. You'll receive:
- Email notification
- Dashboard alert
- SMS (if enabled)

### Creating a Request

1. Click **Maintenance**
2. Click **New Request**
3. Fill in details:
   - Property and unit
   - Category (Plumbing, Electrical, HVAC, etc.)
   - Priority (Low, Medium, High, Urgent)
   - Description
   - Photos
4. Assign to vendor (optional)
5. Click **Submit**

### Managing Requests

#### Status Workflow

1. **New** - Just submitted
2. **In Progress** - Being worked on
3. **On Hold** - Waiting for parts/approval
4. **Completed** - Work finished
5. **Closed** - Verified and billed

#### Assigning Vendors

1. Open maintenance request
2. Click **Assign Vendor**
3. Select from your vendor list or add new
4. Vendor receives email notification
5. Track progress

### Maintenance Reports

View analytics:
- Average response time
- Cost by category
- Most frequent issues
- Vendor performance

## Troubleshooting

### Common Issues

#### Can't Log In

**Problem:** Forgot password or account locked

**Solution:**
1. Click "Forgot Password" on login screen
2. Enter your email
3. Check email for reset link
4. Create new password
5. Try logging in again

If still having issues:
- Clear browser cache
- Try different browser
- Contact support

#### Property/Tenant Not Showing

**Problem:** Data seems missing

**Solution:**
1. Check filters - you may have active filters hiding data
2. Refresh page (F5 or Cmd+R)
3. Verify you're in correct business/portfolio
4. Check "Archived" or "Inactive" sections

#### Payment Not Recorded

**Problem:** Payment entered but not appearing

**Solution:**
1. Check if payment is pending approval
2. Verify correct tenant and date selected
3. Refresh the page
4. Check "All Payments" view (may be filtered)

#### Mobile Display Issues

**Problem:** Page doesn't display correctly on mobile

**Solution:**
1. Update your browser to latest version
2. Clear browser cache
3. Try landscape vs. portrait orientation
4. Use Chrome or Safari for best experience

## Tips & Best Practices

### Organization

1. **Use Consistent Naming** - Standardize property and unit names
2. **Tag Everything** - Add categories and labels for easy searching
3. **Regular Updates** - Keep tenant info and lease dates current
4. **Document Everything** - Upload leases, inspections, photos

### Financial Management

1. **Record Immediately** - Enter payments/expenses same day
2. **Reconcile Monthly** - Match with bank statements

## Glossary

**Business** - For property managers, a client's property portfolio

**Dashboard** - Main overview screen showing key metrics

**Lease** - Rental agreement between landlord and tenant

**Occupancy Rate** - Percentage of units that are rented

**Portfolio** - Collection of properties under one owner

**Security Deposit** - Refundable deposit held for damages

**Unit** - Individual rentable space within a property

**Vacancy** - Empty unit available for rent

**Vendor** - Service provider for maintenance/repairs
